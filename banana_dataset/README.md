# Banana Dataset

Dataset structure for semi-supervised YOLO training.

## Directory Structure

| Folder | Purpose |
|--------|---------|
| `Banana_Dataset_2024_TrainValTest/` | Supervised baseline training |
| `Banana_Dataset_2024_2025/Unlabeled_Images_2025/` | Unlabeled pool for pseudo-labeling |
| `Val_Clean/` | Validation set for model evaluation |
| `Labeled_From_Val/` | Labeled seed data for semi-supervised training |
