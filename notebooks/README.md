# Jupyter Notebooks

This directory contains Jupyter notebooks for data analysis, visualization, and experimentation.

## Usage

- Keep notebooks organized by topic or analysis stage
- Name notebooks with a prefix number for ordering (e.g., `01_data_exploration.ipynb`)
- Document your notebooks with markdown cells

## Best Practices

- Keep code in notebooks clean and well-documented
- Consider moving reusable functions to the `src/` directory
- Export important visualizations to the `docs/` directory
- Restart and run all cells before committing to ensure reproducibility
